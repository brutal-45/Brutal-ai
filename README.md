### ⚡ Installation

```bash
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

git clone https://github.com/brutal-ai/brutal-ai.git

# Navigate to project
cd brutal-ai

# Install dependencies
bun install

# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

bun run dev

# Open in browser
# → http://localhost:3000
```
